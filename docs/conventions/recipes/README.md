# 任务 Recipes

每篇 Recipe 是一类常见开发任务的最小完整示例。照着做就能产出符合架构的代码。

| Recipe | 适用场景 |
|--------|---------|
| [new_mod](new_mod.md) | 新建一个独立 Mod |
| [new_ability](new_ability.md) | 新增一个 GAS 技能 |
| [new_system](new_system.md) | 新增一个 ECS System |
| [new_component](new_component.md) | 新增一个 ECS Component |
| [new_order](new_order.md) | 新增一种交互/命令类型 |
| [new_presenter](new_presenter.md) | 新增一个表现/UI Performer |
| [new_config](new_config.md) | 新增一种配置类型接入 ConfigPipeline |
| [new_trigger](new_trigger.md) | 新增一个事件触发器 |

每篇 Recipe 遵循统一结构：目标 → 文件清单 → 最小代码 → 挂靠点 → 检查清单。
