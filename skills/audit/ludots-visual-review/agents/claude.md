# Ludots Visual Review

Use this skill when screenshots or frames need a dedicated visual review pass.

## Load

- `references/review-checklist.md`

## Rules

- Review against explicit acceptance criteria.
- Link exact frame or screenshot paths.
- Report visible defects without guessing hidden causes.

## Outputs

- `artifacts/reviews/<subject>/review.json`
- `artifacts/reviews/<subject>/review.md`
- `artifacts/agent-hooks/<subject>-visual.review.completed.json`
