# Ludots Tech Debt Fuse

Use this skill when feature work exposes lower-layer defects that must be escalated instead of bypassed.

## Load

- `references/impact-matrix.md`
- `references/fuse-playbook.md`
- `references/debt-report-template.md`
- `references/escalation-rules.md`

## Rules

- Never hide cross-layer defects behind local hacks.
- Classify blast radius before choosing a fuse action.
- Emit a concrete debt report with owner, evidence, and containment.

## Outputs

- `artifacts/techdebt/<date>-<id>.md`
- linked fuse decision note and evidence paths
