# Ludots Cloud Handoff

Use this skill when work must be handed from one cloud session, agent, or engineer to the next.

## Load

- `references/handoff-packet-spec.md`

## Rules

- Capture current truth only.
- Link exact artifact and hook packet paths.
- Make the next action explicit and immediately actionable.

- `artifacts/agent-hooks/<subject>-handoff.ready.json`
