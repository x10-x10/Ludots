# Handoff Packet Spec

Each handoff should include:

- subject key and branch
- completed work with exact paths
- pending blockers with owner or dependency
- next 1-3 actions in execution order
- latest linked hook packets
- latest linked evidence and review artifacts

Preferred output paths:

- `artifacts/handoffs/<subject>/handoff.md`
- `artifacts/agent-hooks/<subject>-handoff.ready.json`
