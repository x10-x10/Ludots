# Ludots Video Frame Extract

Use this skill when a recording must be converted into keyframes and a contact sheet for review.

## Load

- `references/frame-extract-spec.md`

## Rules

- Keep timestamps traceable.
- Extract semantic moments, not arbitrary noise.
- Produce both raw frames and a compact overview when possible.

## Outputs

- `artifacts/evidence/<subject>/frames/`
- `artifacts/evidence/<subject>/contact-sheet.png`
- `artifacts/agent-hooks/<subject>-visual.frames.ready.json`
