# Ludots Visual Capture

Use this skill when a workflow needs screenshots or a recording as first-class evidence.

## Load

- `references/capture-checklist.md`

## Rules

- Capture deterministic scenarios.
- Keep raw artifacts.
- Always write an evidence manifest.

## Outputs

- `artifacts/evidence/<subject>/capture/`
- `artifacts/evidence/<subject>/manifest.json`
- `artifacts/agent-hooks/<subject>-visual.capture.completed.json`
